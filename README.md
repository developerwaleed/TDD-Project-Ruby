# TDD-Project-Ruby
The project was to practice the TDD programming!

## Getting Started
Clone the project

```bash
  git clone https://github.com/developerwaleed/TDD-Project-Ruby.git
```

Go to the project directory

```bash
  cd TDD-Project-Ruby
```

Install dependencies

```bash
  gem install rspec
  gem install rubocop
```

Start the server

```bash
  rubocop -a
  rspec spec
```
